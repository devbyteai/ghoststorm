Algorithm Details

WTF is this screen?

This screen manages the secret codes that let GhostStorm talk to social media platforms like TikTok, Instagram, YouTube, etc.

T TikTok Algorithms

TikTok uses special "signatures" (X-Bogus, X-Gorgon) to block bots. These signatures are like passwords that change every week. We need to keep them updated or TikTok will block our requests.

S Social Platform APIs

Instagram, YouTube, Twitter etc. use "OAuth tokens" - these are like login sessions that let us access their data. They're more stable but expire after 60 days.

Active = Working fine Weekly Update = Needs refresh often Deprecated = Old, don't use

Note: Most users don't need to touch this. It's for debugging when automation stops working.
